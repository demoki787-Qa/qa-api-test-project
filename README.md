## QA API Test Project

Projeto de testes de API utilizando a API pública JSONPlaceholder.

## Objetivo
Demonstrar habilidades em testes manuais de API, documentação de casos de teste, registro de bugs e consultas SQL.

## Ferramentas
- HoppScotch / Postman
- GitHub
- SQL

## Estrutura
- docs → documentação
- test-cases → casos de teste
- bug-reports → registro de bugs
- evidence → evidências dos testes
- sql → consultas SQL

 ## Resultados
- Bug encontrado: post com title vazio
- Todas as queries foram testadas no DB Fiddle
